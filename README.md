# Aerospace Engineering Simulations

A collection of engineering simulations built in Python,
focused on propulsion and aerospace applications.

Built alongside B.E in Mechanical Engineering at 
Bangalore Institute of Technology.

## Projects

| # | Project | Concepts | Status |
|---|---------|----------|--------|
| 01 | [Brayton Cycle Simulator](01-brayton-cycle/) | Basic thermodynamic process in jet engines and gas turbines | Complete |
| 02 | [Rocket Nozzle Flow Calculator](02-rocket-nozzle/) | Thrust and flow simulation through a C-D nozzle | Complete |
| 03 | [Blackbody Radiation Simulator](03-black-body-radiation/) | Planck's Law, Wien's Displacement, and Stefan-Boltzmann radiation | Complete |

## Tools Used
Python, NumPy, Matplotlib, SciPy, Jupyter
